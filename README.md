# Trendoraz 

A premium, high-fashion e-commerce platform and administrative dashboard.

## 🚀 Overview

Trendoraz features a minimalist off-white aesthetic with smooth scroll animations, dynamic collection grids, and a complete admin dashboard for full product management (sizes, colors, materials, fits). 

## 📁 Repository Structure

- `trendoraz-admin/` - The core application including the public-facing e-commerce storefront and administrative dashboard built with **Next.js 15, Neon DB, and Tailwind CSS v4**.

## ✨ Features

- **Premium "Heirloom" Storefront**: Minimalist aesthetic with hero carousels and dynamic collection grids.
- **Dynamic New Arrivals**: Real-time product synchronization from Neon DB.
- **Admin Dashboard**: Full CRUD management specialized for fashion attributes.
- **Responsive Layout**: Optimized for both mobile and desktop.

## 🛠️ Technology Stack

- **Frontend**: Next.js 15 (React 19), TypeScript, Tailwind CSS v4, Lucide React icons.
- **Backend/Database**: Neon DB (PostgreSQL) using `@neondatabase/serverless`.
- **Infrastructure**: Vercel-ready with edge compatibility.

## 💻 Getting Started

To run the application locally, navigate to the `trendoraz-admin` directory and refer to its specific `README.md` for detailed instructions.

```bash
cd trendoraz-admin
npm install
npm run dev
```
